## Prerequisites

You need **Creatio 8.0** or newer, either Cloud or On-Premise. SQL Cor
supports PostgreSQL and MSSQL databases.

## Download

You can download the latest SQL Cor package directly from GitHub:

<a href="https://github.com/OpusCor/SQL/releases/latest" class="opuscor-btn opuscor-btn--primary">
  Download SQL Cor v1.0
</a>

## Next steps

Once installed, see the [User Guide](/v1.0/user-guide/) to start
running queries, or the [Admin Guide](/v1.0/admin-guide/) to configure
access control.
