> [!note] Документація в процесі підготовки
> Повна інструкція з встановлення готується. Ця сторінка опише, як
> встановити пакет SQL Cor у середовища Creatio Cloud та On-Premise,
> включаючи налаштування PostgreSQL та MSSQL.

## Передумови

Вам потрібно **Creatio 8.0** або новіше, Cloud або On-Premise.
SQL Cor підтримує бази даних PostgreSQL та MSSQL.

## Наступні кроки

Після встановлення див. [Посібник користувача](/v1.0/user-guide/),
щоб почати виконувати запити, або
[Посібник адміністратора](/v1.0/admin-guide/), щоб налаштувати
контроль доступу.
