## Quick checks

- Database connection status is shown in the footer (🟢 / 🔴 / 🟡)
- For permission errors, see your access level in
  [Admin Guide → Access Control](/v1.0/admin-guide/#access-control)
- For query errors, check the [Message Reference](/v1.0/reference/messages/)

For unresolved issues, contact your SQL Cor administrator or open an
issue on GitHub.
